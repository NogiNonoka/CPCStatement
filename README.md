# CPCStatement
Generate a XCPC Statement

## How to use

### Titlepage

- Edit `titlepage.tex`

### Problems

- Tempalte file `./problem/template.tex`
- Add Problem files into folder `./problem/`
- Add `\input{./problem/*.tex}` or `\include{./problem/*.tex}` into `statements.tex`

## Build

Build with xeLaTex
